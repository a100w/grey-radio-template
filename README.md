# Grey Radio template
A beautiful laut.fm Radio template for everyone!

# Requirements
- A Webserver (Apache2, Nginx or others..)
- PHP5 or higher

# How to install
- Upload the files to your webserver
- Edit the config.php to your needs
- Done!
